# Contributing to Digital Enlightenment

Thank you for helping expand this knowledge for beginners!

## How to Contribute
1. Fork this repository
2. Create a new branch (feature-newtopic)
3. Add simple, friendly explanations
4. Submit a Pull Request

## Writing Style Rules
- Use simple English  
- Avoid jargon  
- Use visuals, examples, analogies  
- Keep it encouraging and friendly  
- Focus on empowering beginners  

## Safeway Guardian Standard
Every section must reflect the SG mission:
**Serve, Guide, Save, Protect Humanity**
